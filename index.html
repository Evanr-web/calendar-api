☦ UGCC Liturgical Calendar API

Static JSON API serving the Ukrainian Greek-Catholic Church liturgical calendar for 2026. Gregorian (New) calendar.

Endpoints

Full year calendar:
GET /api/2026/calendar.json
Month calendar:
GET /api/2026/{MM}/calendar.json
Individual day (full data + services):
GET /api/2026/{MM}/{DD}.json

Data per day

  • Date, day of week, liturgical tone
  • Fasting level and liturgical season
  • Feast rank (Great Feast / Feast / Polyeleos / Commemoration / Ordinary)
  • Saint commemorations (Ukrainian + English)
  • Troparion and kontakion
  • Rubrics (ustav — service composition instructions)
  • Full service texts: Vespers, Matins, Compline/Hours (Jan–May 2026)

Coverage

  • Calendar metadata: 365/365 days
  • Full service texts: 151/365 days (Jan–May; Jun–Dec as published)
  • English saint names: 365/365 days
  • Troparion/kontakion: 147/365 days
